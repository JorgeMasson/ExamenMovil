## Prueba Móvil

To install:

1. [Yarn](https://yarnpkg.com/getting-started/install) package manager.

## Steps for Local Development

1. Clone the Repo:

```bash
$ git clone https://github.com/JorgeMasson/ExamenMovil.git
```

2. Install dependencies

```bash
$ yarn install
```
3. Run the app

```bash
$ yarn run start
```
